#### 9–14. 记录结果.

改你的计算器程序(练习 5-6 )使之接受命令行参数. 例如: 

    $ calc.py 1 + 2 
    
只输出计算结果. 另外, 把每个表达式和它的结果写入到一个磁盘文件中. 当使用下面的命令时:
 
    $ calc.py print 
     
会把记录的内容显示到屏幕上, 然后重置文件. 这里是样例展示: 

    $ calc.py 1 + 2
    3
    $ calc.py 3 ^ 3
    27
    $ calc.py print
    1 + 2
    3
    3 ^ 3
    27
    $ calc.py print
    $
